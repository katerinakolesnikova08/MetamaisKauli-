# MetamaisKauliņš

Izstrādāts vienkāršs spēles prototips ar nosaukumu “Metamais kauliņš”, kurā lietotājs var mest kauliņu un krāt punktus. Spēle ir izstrādāta Python programmēšanas valodā, izmantojot Tkinter bibliotēku grafiskās lietotāja saskarnes izveidei.

Spēlē ir realizēti visi galvenie dizaina skati: sākuma ekrāns, spēles ekrāns un rezultāta ekrāns. Tika izstrādāts metamo kauliņu mehānisms, kurā tiek ģenerēts nejaušs skaitlis no 1 līdz 6. Spēles laikā tiek veikta punktu skaitīšana, kā arī nodrošināta pāreja starp dažādiem spēles ekrāniem.

Daļa funkcionalitātes netika realizēta vai ir realizēta daļēji. Spēlē nav ieviesta spēlētāja vārda ievade, rezultāti netiek saglabāti failā, nav vairāku spēlētāju režīma, kā arī nav definēti uzvaras vai zaudējuma nosacījumi.

Secinot, izstrādātais prototips veiksmīgi demonstrē spēles pamatloģiku un lietotāja saskarnes struktūru. Spēli nākotnē varētu paplašināt, pievienojot papildu funkcionalitāti, piemēram, rezultātu saglabāšanu, spēlētāja vārda ievadi un sarežģītākus spēles noteikumus.
